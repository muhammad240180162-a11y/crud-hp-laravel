# 📱 CRUD Penjualan HP (Laravel)

Project CRUD sederhana untuk mengelola data kategori HP menggunakan Laravel + MySQL.

---

## 🚀 Fitur

- ✅ Tambah Category (Create)
- ✅ Tampilkan Category (Read)
- ✅ Edit Category (Update)
- ✅ Hapus Category (Delete)

---

## 🛠️ Teknologi

- Laravel
- PHP
- MySQL
- Bootstrap 5
